# STM32-Robot-Project
基于STM32F103的机器人/AGV控制开发项目

## 功能特性
- PWM电机控制
- 编码器速度反馈
- ROS通信接口

## 硬件依赖
- STM32F103ZET6开发板
- 直流电机 + 编码器模块
- ESP8266 WiFi模组

## 快速开始
1. 克隆仓库：`git clone https://github.com/yangyang-dk/STM32-Robot-Project.git`
2. 使用Keil打开工程文件 `project.uvprojx`
3. 编译并烧录到开发板
